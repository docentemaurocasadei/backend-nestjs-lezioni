Creare un progetto NestJS e leggere variabili da .env
Obiettivo

In questo esercizio imparerai a:

creare un nuovo progetto con NestJS

configurare un file .env

installare e usare @nestjs/config

leggere una variabile di ambiente tramite un service

Le variabili .env vengono utilizzate nelle applicazioni backend per configurare:

porte del server

connessioni al database

chiavi API

parametri di configurazione dell’applicazione

Parte 1 – Creazione del progetto
Attività per lo studente

Installa il CLI di NestJS globalmente.

Verifica che il CLI sia installato.

Crea un nuovo progetto NestJS chiamato env-demo.

Entra nella cartella del progetto.

Avvia il server per verificare che il progetto funzioni.
